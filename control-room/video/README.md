# Video Files Directory

Place your video files in this directory:

- `part1.mp4` - First segment of the presentation video
- `part2.mp4` - Second segment of the presentation video

## Video Guidelines

- Format: MP4 (H.264 codec recommended)
- Resolution: 1920x1080 (Full HD) or 1080x1920 (portrait mode)
- File size: Keep under 100MB for optimal loading
- Audio: AAC codec, stereo

## File Names

Make sure your files are named exactly as follows:

- `part1.mp4`
- `part2.mp4`

Or update the paths in `public/display.js` if using different names.
